Collection of various windows & doors from the LPC, plus some recolors

Update 2021-01-01 (version 2): major revision, added many new windows and doors, removed a few trivial recolors, and combined into single spritesheet. Importantly, changed alignment of many of the windows to allow more possible combinations; see [LPC] Victorian Town Decorations for some examples. Many more windows can be found at [LPC] Victorian Buildings as well. This revision is not backwards-compatible with the previous version, so I am leaving those assets in place.
